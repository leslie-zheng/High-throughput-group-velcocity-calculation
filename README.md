Supplementary Data

The attached file contains the mp_id, transverse and longitudinal group velocities, and the number of sites in the primitive cell for all materials included in our screening. All data were generated using the MACE machine-learning interatomic potential.

These data were used solely for screening materials with ultralow lattice thermal conductivity. Note that several extremely large values (in sound velocities or predicted Îº) arise from controlable instabilities and numerical noise in the current MACE model. These outliers do not affect the effectiveness or reliability of our universal descriptor for lattice thermal conductivity, which depends only on coarse phonon features. Importantly, this unstability of current MACE model also did not affect the materials screening process for ultra-low thernal condcutivity.  

This dataset supports the findings presented in the following manuscript:

Shen, X., Zheng, J., Koza, M. M., Levinsky, P., Hejtmanek, J., Boullay, P., ... & Guilmeau, E. (2025).
Accelerated Discovery of Crystalline Materials with Record Ultralow Lattice Thermal Conductivity via a Universal Descriptor.
arXiv:2511.21172.
