Supplementary Data

Attached file including the mp_id, transverse and longitudinal group velocity and nsites in primitice cell for screening materials, all the data obtained using MACE software.

all the data file was as supporting materials for the following paper: 

Shen, X., Zheng, J., Koza, M. M., Levinsky, P., Hejtmanek, J., Boullay, P., ... & Guilmeau, E. (2025). Accelerated Discovery of Crystalline Materials with Record Ultralow Lattice Thermal Conductivity via a Universal Descriptor. arXiv preprint arXiv:2511.21172.
